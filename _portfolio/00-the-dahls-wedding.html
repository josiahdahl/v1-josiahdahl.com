A fully-responsive, hand-coded static site for my wedding. Built using Foundation 6 and semantic SCSS. Highlights include a custom RSVP system built with jQuery that uses Stamplay as a backend.

Mailgun is used for for sending confirmation e-mails and Mailchimp is used for mailing lists.

Design by Nicole Dahl. Coding by Josiah Dahl.
